Mengapa Docker & CI/CD?

Di era cloud computing, kemampuan mengelola container dan automasi deployment adalah skill wajib bagi Software Engineer. Training ini dirancang khusus untuk menjembatani teori akademik dengan **praktik industri**.

  • Hilangkan masalah "It works on my machine"
  • Percepat proses development dengan Isolasi Environment
  • Automasi testing dan deployment dengan Pipeline CI/CD

Materi yang Dipelajari

Kurikulum disusun berdasarkan kebutuhan industri saat ini.

Docker Fundamentals

Arsitektur Container, Image vs Container, Docker CLI, dan Membuat Dockerfile.

Advanced Docker

Persistensi Data dengan Volume & Komunikasi antar service dengan Docker Network.

Orchestration

Menjalankan multi-container applications secara rapi menggunakan Docker Compose.

CI/CD & Deployment

Automasi Build Image, Integration Test, Push Registry, hingga Auto-Deploy ke Server VPS.

Benefit Mengikuti Training

  • Hands-on Real World Project
  • E-Certificate Training
  • Networking dengan Praktisi
